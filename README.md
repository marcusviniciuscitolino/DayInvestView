# DayInvestView

Plataforma de visualização personalizada de dashboards financeiros para investidores de alta renda.

## Tecnologias

- Angular 17
- Angular Material
- Chart.js / ng2-charts
- TypeScript

## Instalação

```bash
npm install
```

## Execução

```bash
npm start
```

A aplicação estará disponível em `http://localhost:4200`

## Estrutura do Projeto

- `/src/app/core` - Serviços, modelos e guards
- `/src/app/shared` - Componentes e módulos compartilhados
- `/src/app/layouts` - Layouts da aplicação
- `/src/app/features` - Módulos de funcionalidades (auth, manager, investor)
- `/src/assets/data` - Dados mockados (usuários, investimentos, temas)
